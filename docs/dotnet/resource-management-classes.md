## <a name="in-this-section"></a>このセクションの内容

[auto_gcroot](../dotnet/auto-gcroot.md)\
仮想ハンドルをネイティブ型に埋め込みます。

[auto_handle](../dotnet/auto-handle.md)\
仮想ハンドルをマネージ型に埋め込みます。
