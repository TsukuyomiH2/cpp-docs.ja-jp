# <a name="omp-pragma"></a>omp プラグマ

省略可能なディレクティブの句と共に、1 つ以上の OpenMP ディレクティブを受け取ります。

## <a name="syntax"></a>構文

> **#pragma omp** *ディレクティブ*

## <a name="remarks"></a>解説

詳細については、「 [OpenMP ディレクティブ](../parallel/openmp/reference/openmp-directives.md)」を参照してください。
