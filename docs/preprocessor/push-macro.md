# <a name="push_macro-pragma"></a>push_macro プラグマ

マクロ *名* マクロの値を、このマクロのスタックの一番上に保存します。

## <a name="syntax"></a>構文

> **#pragma push_macro ("** _マクロ名_ **")**

## <a name="remarks"></a>解説

*マクロ名* の値を取得するには、を使用 `pop_macro` します。

サンプルについては、「 [pop_macro](../preprocessor/pop-macro.md) 」を参照してください。
