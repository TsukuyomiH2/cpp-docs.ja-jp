# <a name="ltcstdiogt"></a>&lt;cstdio&gt;

標準 C ライブラリヘッダーをインクルード \<stdio.h> し、関連する名前を名前空間に追加し `std` ます。

## <a name="syntax"></a>構文

```cpp
#include <cstdio>
```

## <a name="remarks"></a>解説

このヘッダーをインクルードすると、標準 C ライブラリ ヘッダーの外部リンケージで宣言された名前が、`std` 名前空間でも宣言されます。
