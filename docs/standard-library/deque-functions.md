# <a name="ltdequegt-functions"></a>&lt;deque&gt; 関数

## <a name="swap"></a><a name="swap"></a> フォト

2 つの deque の要素を交換します。

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>パラメーター

*左側*\
`deque` 型のオブジェクト。

*そうです*\
`deque` 型のオブジェクト。
