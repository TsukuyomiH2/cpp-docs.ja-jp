# <a name="bad_variant_access-class"></a>bad_variant_access クラス

Bad_variant_access 型のオブジェクトは、variant オブジェクトの値への無効なアクセスを報告するためにスローされます。

## <a name="syntax"></a>構文

```cpp
class bad_variant_access;
```
