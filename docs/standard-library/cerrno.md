# <a name="ltcerrnogt"></a>&lt;cerrno&gt;

C 標準ライブラリヘッダーをインクルード \<errno.h> し、関連する名前を名前空間に追加し `std` ます。 このヘッダーをインクルードすると、C 標準ライブラリヘッダーの外部リンケージを使用して宣言された名前が、名前空間で宣言され `std` ます。

## <a name="syntax"></a>構文

```cpp
#include <cerrno>
```
