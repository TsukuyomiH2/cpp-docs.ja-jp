# <a name="char_traitsltchargt-struct"></a>char_traits&lt;char&gt; 構造体

テンプレート構造体 **char_traits \<CharType>** 型の要素に特殊化した構造体 **`char`** 。

## <a name="syntax"></a>構文

```cpp
template <>
struct char_traits<char>;
```

## <a name="remarks"></a>解説

特殊化により、構造体は、この型のオブジェクトを操作するライブラリ関数を利用でき **`char`** ます。
