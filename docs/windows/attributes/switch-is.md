# <a name="switch_is"></a>switch_is

共用体メンバーを選択する union 判別として機能する式または識別子を指定します。

## <a name="syntax"></a>構文

```cpp
[switch_is]
```

## <a name="remarks"></a>解説

**Switch_is** C++属性には、 [switch_is](/windows/win32/Midl/switch-is) MIDL 属性と同じ機能があります。
