# <a name="unique-c"></a>unique (C++)

一意のポインターを指定します。

## <a name="syntax"></a>構文

```cpp
[unique]
```

## <a name="remarks"></a>解説

**一意**の C++ 属性には、[固有](/windows/win32/Midl/unique)の MIDL 属性と同じ機能があります。

## <a name="example"></a>例

**Unique**の使用例については、「 [ref](ref-cpp.md)の例」を参照してください。
