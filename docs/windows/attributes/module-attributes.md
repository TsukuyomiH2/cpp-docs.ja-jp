# <a name="module-attributes"></a>モジュール属性

次の属性は、 [module](module-cpp.md) 属性にのみ適用できます。

|属性|説明|
|---------------|-----------------|
|[typelib](helpstringdll.md)|ドキュメント文字列参照 (ローカライズ) を実行するために使用する DLL の名前を指定します。|
