# <a name="remainders"></a>Remainders (剰余)

**ANSI 3.3.5** 整数除算の剰余の符号

剰余の符号は、被除数の符号と同じです。 たとえば、オブジェクトに適用された

```
50 / -6 == -8
50 % -6 ==  2
-50 /  6 == -8
-50 %  6 == -2
```
