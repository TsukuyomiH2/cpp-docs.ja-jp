# <a name="domain-errors"></a>ドメイン エラー

**ANSI 4.5.1** ドメイン エラーで数値演算関数が返す値

ERRNO.H ファイルでは、ドメイン エラー定数 `EDOM` は 33 として定義されています。  戻り値の詳細については、エラーが発生した特定の関数のヘルプ トピックをご覧ください。
