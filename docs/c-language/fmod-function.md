# <a name="fmod-function"></a>fmod 関数

**ANSI 4.5.6.4**`fmod`関数の 2 番目の引数がゼロのとき、ドメイン エラーが発生するか、ゼロが返されるか

`fmod` 関数の 2 番目の引数がゼロのとき、関数はゼロを返します。
