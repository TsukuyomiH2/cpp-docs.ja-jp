# <a name="_com_ptr_trelease"></a>_com_ptr_t::Release

**マイクロソフト固有**

カプセル化**Release**されたインターフェイス ポインター`IUnknown`の Release メンバー関数を呼び出します。

## <a name="syntax"></a>構文

```cpp
void Release( );
```

## <a name="remarks"></a>解説

カプセル`IUnknown::Release`化されたインターフェイス ポインターを呼び出`E_POINTER`します。
