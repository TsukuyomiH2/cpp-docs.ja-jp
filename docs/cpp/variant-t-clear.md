# <a name="_variant_tclear"></a>_variant_t::Clear

**Microsoft 固有の仕様**

カプセル化された `VARIANT` オブジェクトをクリアします。

## <a name="syntax"></a>構文

```
void Clear( );
```

## <a name="remarks"></a>解説

カプセル化された `VARIANT` オブジェクトで `VariantClear` を呼び出します。
