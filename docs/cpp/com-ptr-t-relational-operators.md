**Microsoft 固有の仕様**

## <a name="remarks"></a>解説

スマート ポインター オブジェクトを、別のスマート ポインター、生のインターフェイス ポインター、または NULL と比較します。 NULL ポインターテストを除き、これらの演算子はまず `IUnknown`の両方のポインターを照会し、結果を比較します。
