# <a name="_bstr_toperator-"></a>_bstr_t::operator !

**Microsoft 固有の仕様**

カプセル化された `BSTR` が NULL 文字列であるかどうかを確認します。

## <a name="syntax"></a>構文

```
bool operator!( ) const throw( );
```

## <a name="return-value"></a>戻り値

Yes の場合は TRUE、そうでない場合は FALSE を返します。
