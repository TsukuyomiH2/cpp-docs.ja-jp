# <a name="work-with-odbc-and-drivers"></a>ODBC と ODBC ドライバーの使用

ODBC を使用すると、1つのデータベース管理システム (DBMS) だけに依存しないアプリケーションを作成できます。 ユーザーが DBMS 固有の ODBC ドライバーを使用している複数の Dbms で動作するアプリケーションを1つ作成できます。
