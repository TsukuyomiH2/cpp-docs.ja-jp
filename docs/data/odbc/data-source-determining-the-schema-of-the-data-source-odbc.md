# <a name="data-source-determining-the-schema-of-the-data-source-odbc"></a>データ ソース : データ ソースのスキーマの判定 (ODBC)

このトピックの内容は、MFC ODBC クラスに該当します。

`CRecordset` オブジェクトにデータメンバーを設定するには、接続先のデータソースのスキーマを把握しておく必要があります。 データソースのスキーマを決定するには、データソース内のテーブルの一覧、各テーブル内の列の一覧、各列のデータ型、およびインデックスの存在を取得する必要があります。
