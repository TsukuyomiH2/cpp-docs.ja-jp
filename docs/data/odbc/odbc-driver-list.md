# <a name="odbc-driver-list"></a>ODBC ドライバーの一覧

ビジュアルC++には、次のデータベースの ODBC ドライバーが用意されています。

- SQL Server

- Microsoft Access

- Microsoft Excel

- dBASE

- Paradox

- Microsoft Oracle ODBC

- テキスト ファイル

Microsoft およびその他の企業 (ODBC ドライバーパックを含む) から入手できる ODBC ドライバーの詳細については、マイクロソフト製品サポートサービスにお問い合わせください。
