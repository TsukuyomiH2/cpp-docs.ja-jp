フレームワークは、メッセージマップテーブルで受信メッセージとの一致を検索します。 クラスが処理するメッセージごとにメッセージマップエントリを作成し、対応するハンドラーを記述すると、フレームワークは自動的にハンドラーを呼び出します。 次のトピックでは、メッセージマップの検索について説明します。

- [派生メッセージマップ](derived-message-maps.md)
