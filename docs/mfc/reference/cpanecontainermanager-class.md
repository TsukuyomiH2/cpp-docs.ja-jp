フレームワークは、オブジェクトのインスタンスを自動的に作成 `CPaneContainerManager` し、それらを [Cpanedivider クラス](../../mfc/reference/cpanedivider-class.md) オブジェクトまたは [CMultiPaneFrameWnd クラス](../../mfc/reference/cmultipaneframewnd-class.md) オブジェクトに埋め込みます。

クラスは、 `CPaneContainerManager` [Cpanecontainer](../../mfc/reference/cpanecontainer-class.md) オブジェクトから構築されたバイナリツリーのルートへのポインターを格納します。
