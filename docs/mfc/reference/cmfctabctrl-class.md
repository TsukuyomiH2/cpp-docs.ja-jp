# <a name="cmfctabctrl-class"></a>CMFCTabCtrl Class

この`CMFCTabCtrl`クラスは、タブ コントロールの機能を提供します。 タブ コントロールは、上または下にフラットまたは 3D のタブを持つ、ドッキング可能なウィンドウを表示します。 タブにはテキストとイメージを表示でき、アクティブな状態のときに色を変更することもできます。

## <a name="syntax"></a>構文

```
class CMFCTabCtrl : public CMFCBaseTabCtrl
```

## <a name="remarks"></a>解説

この`CMFCTabCtrl`クラスは次の機能をサポートしています。

- 3D、フラット、フラットを含むタブ コントロール スタイルで、水平スクロール バーを共有します。

- ウィンドウの上部または下部に表示されるタブ。

- テキスト、イメージ、またはテキストとイメージを表示するタブ。

- タブがアクティブなときに色を変更するタブ。

- 調整可能なタブの境界線サイズが変更されます。

- 取り外し可能なタブ付きウィンドウ。

この`CMFCTabCtrl`クラスはダイアログ ボックスで使用できますが、Microsoft Excel や Visual Studio などのドッキング コントロール バーを使用するアプリケーションを対象としています。 詳細については、「 [CDockablePane クラス](../../mfc/reference/cdockablepane-class.md)」を参照してください。

サイズ変更可能なドッキング タブ コントロールをアプリケーションに追加するには、次の手順を実行します。

1. [CTabbedPane クラス](../../mfc/reference/ctabbedpane-class.md)のインスタンスを作成します。

1. 呼び出し[CDockable ペイン::作成](../../mfc/reference/cdockablepane-class.md#create).

1. 新しい[タブを追加するには、CBaseTabbedPane を使用します。::AddTab](../../mfc/reference/cbasetabbedpane-class.md#addtab)または[CMFCBaseTabCtrl::挿入タブ](../../mfc/reference/cmfcbasetabctrl-class.md#inserttab)を使用します。

1. [CBasePane::現在](../../mfc/reference/cbasepane-class.md#enabledocking)のドッキング タブ コントロールがメイン フレーム ウィンドウにドッキングできるようにドッキングを呼び出します。

1. [CFrameWndEx::DockPane](../../mfc/reference/cframewndex-class.md#dockpane)を呼び出して、タブ付きウィンドウをメイン フレームにドッキングします。

ドッキング コントロール バーとしてタブ付きウィンドウを作成する方法の例については、「 [CTabbedPane クラス](../../mfc/reference/ctabbedpane-class.md)」を参照してください。 非ドッキング`CMFCTabCtrl`コントロールとして使用するには、オブジェクトを`CMFCTabCtrl`作成し[、CMFCTabCtrl::作成](#create)を呼び出します。

## <a name="cmfctabctrlcreate"></a><a name="create"></a>作成

タブ コントロールを作成し、オブジェクトに`CMFCTabCtrl`アタッチします。

```
BOOL Create(
    Style style,
    const RECT& rect,
    CWnd* pParentWnd,
    UINT nID,
    Location location=LOCATION_BOTTOM,
    BOOL bCloseBtn=FALSE);
```

### <a name="parameters"></a>パラメーター

*style*<br/>
[in]タブ コントロールのスタイル。 詳細については、「解説」を参照してください。

*Rect*<br/>
[in]タブ コントロールに境界を設定する四角形。

*pParentWnd*<br/>
[in]親ウィンドウへのポインター。 NULL にすることはできません。

*nID*<br/>
[in]タブ コントロールの ID。

*location*<br/>
[in]タブの場所。 既定値は LOCATION_BOTTOM です。 詳細については、「解説」を参照してください。

|STYLE_FLAT_SHARED_HORZ_SCROLL|2 次元タブを表示します。 同時に表示できるタブ数が多い場合は、タブ領域の端にスクロール矢印が表示されます。|

## <a name="cmfctabctrlondragenter"></a><a name="ondragenter"></a>CMFCタブCtrl:オンドラッグエンター

カーソルが最初に現在のタブ コントロールのウィンドウに入ったときに、ドラッグ アンド ドロップ操作中にフレームワークによって呼び出されます。

```
virtual DROPEFFECT OnDragEnter(
    COleDataObject* pDataObject,
    DWORD dwKeyState,
    CPoint point);
```

### <a name="parameters"></a>パラメーター

*オブジェクト*<br/>
[in]ユーザーがドラッグするデータを含むデータ オブジェクトへのポイント。

*州*<br/>
[in]修飾子キーの状態を格納します。 このパラメーターは、MK_CONTROL、MK_SHIFT、MK_ALT、MK_LBUTTON、MK_MBUTTON、およびMK_RBUTTONのビットごとの組み合わせ (OR) です。 詳細については、「[マウス入力について](/windows/win32/inputdev/about-mouse-input)」の **「メッセージ パラメータ**」セクションを参照してください。

*ポイント*<br/>
[in]クライアント座標でのカーソルの現在の位置を格納します。

### <a name="return-value"></a>戻り値

常にDROPEFFECT_NONE、つまり、ドロップターゲットはデータを受け入れることができません。

### <a name="remarks"></a>解説

このメソッドは、ドラッグ アンド ドロップ操作をサポートするために使います。 独自のカスタム動作を実装するには、このメソッドをオーバーライドします。

既定では、このメソッドは、`CMFCTabCtrl::OnDragOver`を呼び出すだけで、常に DROPEFFECT_NONE を返します。
