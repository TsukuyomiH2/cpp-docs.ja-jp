# <a name="using-cheaderctrl"></a>CHeaderCtrl の使い方

[クラス CHeaderCtrl](../mfc/reference/cheaderctrl-class.md)で表されるヘッダー コントロールを使用して、列リストの列ヘッダーを表示します。 たとえば、ヘッダー コントロールは、スプレッドシートに列コントロールを実装する場合に便利です。

ヘッダー コントロールは、通常、"ヘッダー項目" と呼ばれる部分に分割され、各部分には、テキストまたは数値の関連する列のタイトルが付けられます。 設定したスタイルに応じて、ユーザーがヘッダー項目を操作するためのさまざまな直接的な方法を提供できます。

> [!NOTE]
> [CListCtrl](../mfc/reference/clistctrl-class.md)埋め込みヘッダー コントロールを提供し[、CListView](../mfc/reference/clistview-class.md)は MFC クラスにカプセル化します`CListCtrl`。 一般に、自分で`CHeaderCtrl`作成するリストにラベルを付けるために使用することを考えます。
