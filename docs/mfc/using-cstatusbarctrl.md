# <a name="using-cstatusbarctrl"></a>CStatusBarCtrl の使い方

ステータスバーコントロール ([CStatusBarCtrl](../mfc/reference/cstatusbarctrl-class.md)) を使用して、アプリケーションに関するさまざまな種類のステータス情報を反映したコントロールウィンドウを作成できます。 ステータスウィンドウは、複数の種類の情報を表示する部分に分割できます。
