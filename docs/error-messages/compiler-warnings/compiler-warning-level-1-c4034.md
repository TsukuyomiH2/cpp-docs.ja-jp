# <a name="compiler-warning-level-1-c4034"></a>コンパイラの警告 (レベル 1) C4034

sizeof は0を返します

演算子は、 **`sizeof`** サイズゼロ (空の構造体、共用体、クラス、または列挙型、または型) のオペランドに適用され **`void`** ます。
