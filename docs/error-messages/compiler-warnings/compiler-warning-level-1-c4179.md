# <a name="compiler-warning-level-1-c4179"></a>コンパイラの警告 (レベル 1) C4179

> '//* ': '/' および '/' として解析され \* ます: 標準のコメント '//' と混同します。

__//\*__ が正しくないコメントの区切り記号です。 __//__ 代わりに、またはを使用 __/\*__ してください。
