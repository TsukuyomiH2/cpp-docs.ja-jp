# <a name="compiler-warning-level-1-c4179"></a>コンパイラの警告 (レベル 1) C4179

> '//* ': '/' および '/\*' として解析されます: 標準のコメント '//' と混同します。

__//\*__ は、正しくないコメントの区切り記号です。 代わりに、 __//__ または __/\*__ を使用してください。
