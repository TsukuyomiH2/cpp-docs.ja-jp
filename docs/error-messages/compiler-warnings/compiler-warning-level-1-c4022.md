# <a name="compiler-warning-level-1-c4022"></a>コンパイラの警告 (レベル 1) C4022

' function ': 実際のパラメーター ' number ' とポインターが一致していません。

実際のパラメーターのポインター型は、対応する仮パラメーターのポインター型と異なります。 実際のパラメーターは変更なしで渡されます。
