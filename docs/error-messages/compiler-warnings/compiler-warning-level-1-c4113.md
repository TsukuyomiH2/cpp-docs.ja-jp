# <a name="compiler-warning-level-1-c4113"></a>コンパイラの警告 (レベル 1) C4113

' identifier1 ' は、' identifier2 ' のパラメーターリストで異なります

関数ポインターが別の関数ポインターに割り当てられていますが、関数の仮パラメーターリストが一致していません。 割り当ては、変更せずにコンパイルされます。
