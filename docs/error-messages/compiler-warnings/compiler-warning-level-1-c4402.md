# <a name="compiler-warning-level-1-c4402"></a>コンパイラの警告 (レベル 1) C4402

PTR 演算子を使用する必要があります

型は、インラインアセンブリコードで型を参照またはキャストするときに、PTR 演算子のないオペランドで使用されます。

コンパイラは、PTR 演算子を前提としています。
