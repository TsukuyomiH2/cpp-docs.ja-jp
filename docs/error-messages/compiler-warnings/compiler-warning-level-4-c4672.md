# <a name="compiler-warning-level-4-c4672"></a>コンパイラの警告 (レベル 4) C4672

'identifier1' があいまいです。 1 つ目を 'identifier2' とします。

ブロックでスローされるように指定されたオブジェクト **`try`** があいまいです。 オブジェクトがスローされる場合、そのオブジェクトを特定できません。
