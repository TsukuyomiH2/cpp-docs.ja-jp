# <a name="compiler-warning-level-1-c4089"></a>コンパイラの警告 (レベル 1) C4089

' function ': 実際のパラメーター ' number '、仮パラメーター ' number ' の型が異なります。

対応する仮パラメーターと実パラメーターの型が異なります。 実際のパラメーターは変更なしで渡されます。 関数は、実際のパラメーターを関数定義で指定された型にキャストします。
