# <a name="compiler-warning-level-1-c4143"></a>コンパイラの警告 (レベル 1) C4143

'same_seg' プラグマはサポートされていません ; 代わりに __based を使用してください。

**#pragma same_seg** は現在サポートされていません。 代わりに [__based](../../cpp/based-pointers-cpp.md) キーワードを使用します。
