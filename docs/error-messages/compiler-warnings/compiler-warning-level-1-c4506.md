# <a name="compiler-warning-level-1-c4506"></a>コンパイラの警告 (レベル 1) C4506

インライン関数 ' function ' の定義がありません。

指定された関数が宣言され、インライン展開のマークが付けられましたが、定義されていません。

コンパイラは関数をインライン化しませんでした。

インライン化する外部関数が `extern` キーワードを使用して宣言されていることを確認します。
