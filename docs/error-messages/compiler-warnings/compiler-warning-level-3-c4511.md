# <a name="compiler-warning-level-3-c4511"></a>コンパイラの警告 (レベル 3) C4511

' class ': コピーコンストラクターを生成できませんでした

コンパイラは、クラスの既定のコピーコンストラクターを生成できませんでした。基底クラスには、プライベートコピーコンストラクターを含めることができます。
