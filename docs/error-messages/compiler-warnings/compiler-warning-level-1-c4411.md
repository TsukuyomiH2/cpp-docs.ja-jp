# <a name="compiler-warning-level-1-c4411"></a>コンパイラの警告 (レベル 1) C4411

' identifier ': シンボルはディスプレイスメントレジスタに解決されます

識別子は、ディスプレイスメントレジスタに解決されるローカルシンボルであるため、別のシンボルを持つオペランドで使用できます。
