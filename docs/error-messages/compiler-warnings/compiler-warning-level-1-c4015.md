# <a name="compiler-warning-level-1-c4015"></a>コンパイラの警告 (レベル 1) C4015

' identifier ': ビットフィールドの型は整数でなければなりません

ビットフィールドは整数型として宣言されていません。 コンパイラは、ビットフィールドの基本型が符号なしであると想定しています。 ビットフィールドは、符号なし整数型として宣言する必要があります。
