# <a name="compiler-warning-level-1-c4406"></a>コンパイラの警告 (レベル 1) C4406

ディレクティブのオペランドが無視されました

ディレクティブはオペランドを受け取りませんが、オペランドが指定されました。

コンパイラは、指定されたオペランドを無視しました。
