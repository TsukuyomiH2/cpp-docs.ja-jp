# <a name="compiler-warning-level-3-c4133"></a>コンパイラの警告 (レベル 3) C4133

' type ': ' type1 ' から ' type1 ' への互換性のない型です

この警告は、異なる型の2つのポインターを減算しようとした場合に発生することがあります。

この警告を回避するには、適切な型キャストを指定します。
