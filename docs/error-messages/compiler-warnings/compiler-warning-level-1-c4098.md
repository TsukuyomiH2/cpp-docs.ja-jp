# <a name="compiler-warning-level-1-c4098"></a>コンパイラの警告 (レベル 1) C4098

' function ': void 関数は値を返します。

戻り値の型が[void](../../cpp/void-cpp.md)で宣言された関数には、 **`return`** 値を返すステートメントがあります。 コンパイラは、関数が型の値を返すことを前提としてい **`int`** ます。
