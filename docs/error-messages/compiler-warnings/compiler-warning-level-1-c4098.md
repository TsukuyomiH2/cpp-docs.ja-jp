# <a name="compiler-warning-level-1-c4098"></a>コンパイラの警告 (レベル 1) C4098

' function ': void 関数は値を返します。

戻り値の型が[void](../../cpp/void-cpp.md)で宣言された関数には、値を返す `return` ステートメントがあります。 コンパイラは、関数が `int`型の値を返すと想定しています。
