# <a name="compiler-warning-level-1-c4226"></a>コンパイラの警告 (レベル 1) C4226

> 非標準の拡張機能が使用されています: '*keyword*' は廃止されたキーワードです

現在のバージョンの Visual C++ は、このキーワードを使用しません。

この警告は、自動的にエラーになります。
