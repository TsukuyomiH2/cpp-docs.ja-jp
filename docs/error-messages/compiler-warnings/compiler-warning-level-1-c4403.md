# <a name="compiler-warning-level-1-c4403"></a>コンパイラの警告 (レベル 1) C4403

無効な PTR 演算子

PTR 演算子がインラインアセンブラーコードで不適切に使用されています。

コンパイラは PTR 演算子を無視しました。
