# <a name="compiler-warning-level-2-c4007"></a>コンパイラの警告 (レベル 2) C4007

' function ': ' attribute ' でなければなりません

関数に必須の属性は明示的に指定されていません。 たとえば、 **main**関数には属性が必要 **`__cdecl`** です。 コンパイラは、属性を強制的に実行します。
