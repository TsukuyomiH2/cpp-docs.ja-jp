# <a name="compiler-warning-level-1-c4164"></a>コンパイラの警告 (レベル 1) C4164

'identifier': 組み込み関数は宣言されていません

指定された組み込み関数は宣言されていません。#include を使用して適切なヘッダー ファイルをインクルードする必要があります。
