# <a name="compiler-error-c3180"></a>コンパイラエラー C3180

' type name ': 名前がメタデータの上限である ' limit ' 文字を超えています

コンパイラにより、メタデータ内のマネージ型の名前が切り捨てられました。 切り捨てにより、型がディレクティブで使用 `#using` できなくなります (または別の言語の同等のもの)。

型名の制限には、任意の名前空間の要件が含まれます。
