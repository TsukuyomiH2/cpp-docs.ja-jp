# <a name="compiler-error-c2795"></a>コンパイラエラー C2795

' super:: function ' はメンバー関数ではありません

このエラーメッセージは、メンバー関数以外のメンバーに[スーパー](../../cpp/super.md)ユーザーがアクセスしようとしたときに表示されます。
