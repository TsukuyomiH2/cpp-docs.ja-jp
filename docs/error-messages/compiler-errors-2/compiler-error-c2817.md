# <a name="compiler-error-c2817"></a>コンパイラ エラー C2817

' operator delete ' の戻り値の型は ' void ' でなければなりません

オーバーロードされた [operator delete](../../standard-library/new-operators.md#op_delete) 関数は値を返すことができません。
