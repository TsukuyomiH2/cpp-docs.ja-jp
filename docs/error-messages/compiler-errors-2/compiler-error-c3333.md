# <a name="compiler-error-c3333"></a>コンパイラ エラー C3333

'type library': 壊れたタイプ ライブラリは #import できません

`#import` ステートメントで指定されたタイプ ライブラリは、コンパイラで読み取ることができません。 可能な場合はタイプ ライブラリを再生成するか、サプライヤーに新しいタイプ ライブラリを要求してください。 Visual C++ に付属の OLE Viewer を使用すると、タイプ ライブラリ ファイルを表示して問題点を確認できます。
