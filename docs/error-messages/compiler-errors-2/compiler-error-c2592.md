'class': 'base_class_2' は、'base_class_1' から継承されるので、再指定することはできません

他の基底クラスから継承していない基底クラスのみを指定できます。 この場合、 `base_class_1` は **`class`** `base_class_1` 既にを継承しているので、の指定にのみ必要です `base_class_2` 。
