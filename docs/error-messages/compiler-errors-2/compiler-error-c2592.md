# <a name="compiler-error-c2592"></a>コンパイラ エラー C2592

'class': 'base_class_2' は、'base_class_1' から継承されるので、再指定することはできません

他の基底クラスから継承していない基底クラスのみを指定できます。 この場合は、`base_class_1` が既に `class` を継承しているため、`base_class_1` の指定で必要になるのは `base_class_2` のみです。
