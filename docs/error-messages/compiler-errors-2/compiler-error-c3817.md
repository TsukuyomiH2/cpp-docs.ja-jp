# <a name="compiler-error-c3817"></a>コンパイラ エラー C3817

' 宣言 ': プロパティは関数にのみ適用できます

**`property`** キーワードは、関数定義にのみ適用できます。

C3817 は、互換性のために残されているコンパイラオプション **/clr: oldSyntax**を使用してのみ到達可能です。
