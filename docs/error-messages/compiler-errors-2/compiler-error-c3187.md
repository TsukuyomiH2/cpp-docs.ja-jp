# <a name="compiler-error-c3187"></a>コンパイラ エラー C3187

> '&#95;&#95;&#95;func&#95;': 関数の本体内でのみ使用できます

定義済みのマクロは、関数本体の外部で使用できません。

このエラーを修正するには、マクロを関数の本体の内部に移動します。
