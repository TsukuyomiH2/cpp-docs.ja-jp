# <a name="compiler-error-c3135"></a>コンパイラエラー C3135

' property ': プロパティに ' const ' または ' volatile ' の型を含めることはできません

[Const](../../cpp/const-cpp.md)キーワードと[volatile](../../cpp/volatile-cpp.md)キーワードは、プロパティでは許可されていません。
