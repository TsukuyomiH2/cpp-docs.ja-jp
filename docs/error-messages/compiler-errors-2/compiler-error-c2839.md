# <a name="compiler-error-c2839"></a>コンパイラ エラー C2839

オーバーロードされた ' operator-> ' の戻り値の型 ' type ' が無効です

演算子は、 `->` クラス、構造体、共用体、またはそのいずれかへの参照を返す必要があります。
