# <a name="compiler-error-c2656"></a>コンパイラエラー C2656

' function ': 関数はビットフィールドとして使用できません

関数は、ビットフィールドのメンバーとして宣言されています。

このエラーは、コンストラクター初期化子リストの構文エラーが原因で発生する可能性があります。
