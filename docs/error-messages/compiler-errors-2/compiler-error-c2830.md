# <a name="compiler-error-c2830"></a>コンパイラ エラー C2830

' operator new ' への配置パラメーターのみに既定値を指定できます

[Operator new](../../standard-library/new-operators.md#op_new)の標準の仮パラメーターに既定値を指定することはできません。 既定値を指定できるのは、ユーザー定義の配置パラメーターだけです。
