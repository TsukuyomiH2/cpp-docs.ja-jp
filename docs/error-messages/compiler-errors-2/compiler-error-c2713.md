# <a name="compiler-error-c2713"></a>コンパイラエラー C2713

関数ごとに許可されている例外処理の形式は1つだけです

同じ関数で構造化例外処理 ( `__try` / **`__except`** ) と C++ 例外処理 ( **`try`** / **`catch`** ) を使用することはできません。
