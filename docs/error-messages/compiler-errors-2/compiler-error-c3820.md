# <a name="compiler-error-c3820"></a>コンパイラ エラー C3820

> '*name*': 初期化子はマネージドでなければなりません

## <a name="remarks"></a>解説

マネージド型の *name* 初期化子は、同様にマネージドにする必要があります。 このエラーを修正するには、マネージ初期化子メソッドを指定します。
