# <a name="compiler-error-c2721"></a>コンパイラエラー C2721

' 指定子 ': ストレージクラス指定子が演算子キーワードと型の間で正しくありません

ユーザー定義型の変換はすべてのストレージクラスに適用されるため、型変換でストレージクラスを指定することはできません。
