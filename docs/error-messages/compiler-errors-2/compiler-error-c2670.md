# <a name="compiler-error-c2670"></a>コンパイラエラー C2670

' identifier ': 関数テンプレートは、パラメーター番号を型 ' type ' から変換できません

パラメーターを必要な型に変換できませんでした。

このエラーは、明示的な変換を作成した場合に修正される可能性があります。
