# <a name="compiler-error-c2517"></a>コンパイラ エラー C2517

' identifier ': ':: ' の右側が定義されていません

スコープ解決演算子 (`::`) の右側にある識別子は、左側のクラス、構造体、または共用体の定義済みのメンバーである必要があります。 クラス、構造体、または共用体の名前が指定されていない場合は、右側の識別子をグローバルスコープで宣言する必要があります。
