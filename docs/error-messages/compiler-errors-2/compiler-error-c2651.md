# <a name="compiler-error-c2651"></a>コンパイラエラー C2651

' data type ': ' operator ' の左側はクラス、構造体、または共用体でなければなりません

テンプレートパラメーターをクラスとして使用するには、クラステンプレートを整数型ではなくクラスで特殊化します。
