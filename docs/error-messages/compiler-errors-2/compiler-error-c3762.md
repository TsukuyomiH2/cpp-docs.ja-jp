# <a name="compiler-error-c3762"></a>コンパイラ エラー C3762

属性 ' attribute ' を処理できません

から継承されたユーザー定義の属性 `System.Security.Permissions.SecurityAttribute` は、セキュリティ属性を定義するために使用されています。 このような属性は、定義されているのと同じアセンブリでは使用できません。
