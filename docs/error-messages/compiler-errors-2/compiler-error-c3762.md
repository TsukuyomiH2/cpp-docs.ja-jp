# <a name="compiler-error-c3762"></a>コンパイラ エラー C3762

属性 ' attribute ' を処理できません

`System.Security.Permissions.SecurityAttribute` から継承したユーザー定義の属性を使用して、セキュリティ属性を定義します。 このような属性は、定義されているのと同じアセンブリでは使用できません。
