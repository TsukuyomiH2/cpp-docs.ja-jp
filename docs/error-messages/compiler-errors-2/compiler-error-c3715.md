# <a name="compiler-error-c3715"></a>コンパイラ エラー C3715

' pointer ': ' class ' へのポインターでなければなりません

[__Hook](../../cpp/hook.md)にポインターを指定したか、有効なクラスを指していない[__unhook](../../cpp/unhook.md)しました。 このエラーを解決するには、`__hook` と `__unhook` の呼び出しで、有効なクラスへのポインターが指定されていることを確認します。
