# <a name="cvtres-warning-cvt4001"></a>CVTRES の警告 CVT4001

> コンピューターの種類が指定されていません。想定される *型*

CVTRES がマシン仕様を見つけられませんでした。 指定されたコンピューターの種類を想定しています。 既定値が正しくない場合は、/MACHINE オプションを使用して CVTRES を再実行します。
