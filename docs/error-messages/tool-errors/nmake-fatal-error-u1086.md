# <a name="nmake-fatal-error-u1086"></a>NMAKE の致命的なエラー U1086

**推論規則に依存関係を含めることはできません**

推論規則のコロン ( **:** ) の後には、次のいずれかを指定する必要があります。

- 改行文字

- セミコロン ( **;)。** コマンドの後に指定できます。

- シャープ記号 ( **#** )。その後にコメントを付けることができます。
