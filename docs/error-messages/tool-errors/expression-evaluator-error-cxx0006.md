# <a name="expression-evaluator-error-cxx0006"></a>式エバリュエーター エラー CXX0006

左かっこがありません

式に、不均衡かっこが見つかりました。 均衡かっこを使用して式を再入力します。

このエラーは CAN0006 と同じです。
