# <a name="math-error-m6203"></a>数値演算エラー M6203

' function ': _OVERFLOW エラー

指定された関数の結果が大きすぎて表すことができませんでした。

このエラーは、関数名、引数、およびエラーの種類を使用して `_matherr` 関数を呼び出します。 `_matherr` 関数を書き直して、特定の実行時の浮動小数点演算エラーの処理をカスタマイズすることができます。
