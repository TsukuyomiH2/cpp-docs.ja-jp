# <a name="resource-compiler-error-rc2015"></a>リソース コンパイラ エラー RC2015

定数に文字が多すぎます

文字定数に3文字を超える文字が含まれています。

文字定数は、1文字 (標準文字定数) または2文字 (長い文字定数) に制限されます。

エスケープシーケンス (たとえば、タブの場合は \t) が1文字に変換されることに注意してください。
