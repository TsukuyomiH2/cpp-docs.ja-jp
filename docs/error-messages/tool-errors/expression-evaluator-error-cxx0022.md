# <a name="expression-evaluator-error-cxx0022"></a>式エバリュエーター エラー CXX0022

_main 前の関数呼び出し

C 式エバリュエーターは、デバッガーが関数 **_main**に入力する前に関数を評価することはできません。 **_Main**が呼び出されるまで、プログラムは適切に初期化されません。

このエラーは CAN0022 と同じです。
