# <a name="bscmake-error-bk1504"></a>BSCMAKE エラー BK1504

ファイル ' filename ' に配置できません [: reason]

BSCMAKE で、ファイル内の場所に移動できませんでした。

### <a name="to-fix-by-checking-the-following-possible-causes"></a>次のような原因をチェックして問題を解決するには

1. フルディスク。

1. ハードウェアエラー。

1. .Sbr ファイルの作成中に、ディスク領域が不足しているか、コンパイラが中断したため、ファイルが切り捨てられました。
