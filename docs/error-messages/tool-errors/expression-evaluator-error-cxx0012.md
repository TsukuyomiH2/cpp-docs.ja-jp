# <a name="expression-evaluator-error-cxx0012"></a>式エバリュエーター エラー CXX0012

左中かっこがありません

式に、不均衡な中かっこが含まれています。 均衡中かっこを使用して式を再入力します。

このエラーは CAN0012 と同じです。
