# <a name="nmake-fatal-error-u1076"></a>NMAKE の致命的なエラー U1076

名前が長すぎます

文字列が次の制限のいずれかを超えています:

- マクロ名には1024文字が使用できます。

- ターゲットパス名に256文字。

- コマンドの場合は2048文字。
