# <a name="bscmake-error-bk1516"></a>BSCMAKE エラー BK1516

bscfile が破損しています。増分更新できません

.Bsc ファイルが破損しています。ビルド中にシステムエラーが発生した可能性があります。 .Bsc ファイルを削除し、すべての .sbr ファイルをリビルドしてから、.bsc ファイルをリビルドします。
