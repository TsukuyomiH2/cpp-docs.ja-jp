# <a name="expression-evaluator-error-cxx0059"></a>式エバリュエーター エラー CXX0059

左オペランドが関数名ではありません

関数呼び出しの左オペランドはクラス名でしたが、関数呼び出しに解決できませんでした。 式でメンバー関数の名前を省略すると、このエラーが発生する可能性があります。

このエラーは CAN0059 と同じです。
