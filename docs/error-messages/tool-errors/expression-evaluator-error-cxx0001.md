# <a name="expression-evaluator-error-cxx0001"></a>式エバリュエーター エラー CXX0001

ユーザー関数の実行中に例外が発生した

実行中のコードにより、一般的な保護エラーが発生しました。 このエラーは CAN0001 と同じです。
