# <a name="nmake-fatal-error-u1083"></a>NMAKE の致命的なエラー U1083

ターゲットマクロ ' target ' は、何も展開されません

指定されたターゲットは、定義されていない、または null 値を持つマクロの呼び出しです。 NMAKE は null ターゲットを処理できません。
