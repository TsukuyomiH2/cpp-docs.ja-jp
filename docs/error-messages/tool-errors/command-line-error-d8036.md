# <a name="command-line-error-d8036"></a>コマンド ライン エラー D8036

'/option ' は複数のソースファイルで許可されていません

これらのコンパイラオプションは、複数のソースファイルでは使用できません。

- アセンブリファイルリストの名前を指定する (/Fa)

- オブジェクトファイル名の変更 (/Fo)

- ローカル変数を使用せずにソースブラウザー情報を作成する (/Fr)

- ローカル変数を含むソースブラウザー情報を作成する (/FR)
