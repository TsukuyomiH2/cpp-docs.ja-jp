# <a name="expression-evaluator-error-cxx0069"></a>式エバリュエーター エラー CXX0069

変数にはスタックフレームが必要です

式エバリュエーターは、スタックフレーム内に存在しないため、変数を評価できません。 これは、インライン関数の一部として宣言された変数によって発生することがあります。
