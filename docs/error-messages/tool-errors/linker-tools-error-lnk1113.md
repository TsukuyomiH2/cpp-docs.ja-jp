# <a name="linker-tools-error-lnk1113"></a>リンカ ツール エラー LNK1113

コンピューターの種類の種類が無効です

オブジェクトヘッダーで指定されたコンピューターの種類が無効です。

このエラーは、ファイルが壊れている場合に発生する可能性があります。 オブジェクトを再構築します。
