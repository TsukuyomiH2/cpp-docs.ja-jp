# <a name="linker-tools-error-lnk1287"></a>リンカ ツール エラー LNK1287

無効なマネージドエントリポイント関数

エントリポイントは、管理対象イメージに対して有効ではありません。 マネージエントリポイント関数の戻り値の型には、`void` または `int`のみを指定できます。 マネージエントリポイント関数のパラメーターの型は、`void` または `String []`のみを指定できます。
