# <a name="profile-guided-optimization-warning-pg1039"></a>ガイド付き最適化のプロファイルの警告 PG1039

スイープに時間がかかるまで待機できませんでした。

アプリケーションで [pgosweep](../../build/pgosweep.md) の実行が許可されていませんでした。  可能であれば、アプリケーションがアイドル状態のときに pgosweep を実行します。
