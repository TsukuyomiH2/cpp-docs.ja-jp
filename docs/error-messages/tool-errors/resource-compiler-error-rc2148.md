# <a name="resource-compiler-error-rc2148"></a>リソース コンパイラ エラー RC2148

サブ言語 ID が大きすぎます

サブ言語 ID の値が範囲外です。

**LANGUAGE** ステートメントでは次の構文を使用しなければなりません。

**言語** *primary_language_ID*、*secondary_language_ID*

有効なサブ言語 Id は、WINNT.H ファイル内の**SUBLANG_** 定数として定義されています。
