# <a name="linker-tools-warning-lnk4220"></a>リンカー ツールの警告 LNK4220

' リンカーオプション ' の値 ' value ' が無効です。想定される既定値

範囲外の値が [/TLBID](../../build/reference/tlbid-specify-resource-id-for-typelib.md) オプションと共に指定されました。 **/TLBID** の既定値は1です。
