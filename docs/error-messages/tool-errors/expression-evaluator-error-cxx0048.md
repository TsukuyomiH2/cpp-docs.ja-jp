# <a name="expression-evaluator-error-cxx0048"></a>式エバリュエーター エラー CXX0048

呼び出しシーケンスはサポートされていません

式で指定された関数が、C 式エバリュエーターでサポートされていない呼び出しシーケンスを使用しています。 ウォッチウィンドウ式では、この関数を呼び出すことはできません。

このエラーは CAN0048 と同じです。
