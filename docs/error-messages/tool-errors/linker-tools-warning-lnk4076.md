# <a name="linker-tools-warning-lnk4076"></a>リンカー ツールの警告 LNK4076

インクリメンタルステータスファイル ' filename ' が無効です。フルのリンク

リンクは、増分ステータス (.ilk) ファイルに書き込むことができません。 `filename` が破損しているか、インクリメンタルリンクデータベースではありません。 ファイルを削除して再リンクします。
