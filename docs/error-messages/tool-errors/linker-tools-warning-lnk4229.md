# <a name="linker-tools-warning-lnk4229"></a>リンカー ツールの警告 LNK4229

無効なディレクティブ/ディレクティブが見つかりました。無効

[Comment](../../preprocessor/comment-c-cpp.md)プラグマを使用して渡されたディレクティブが無効です。 リンカーは `/directive` を無視します。
