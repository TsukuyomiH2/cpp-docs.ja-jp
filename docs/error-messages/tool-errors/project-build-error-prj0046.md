# <a name="project-build-error-prj0046"></a>プロジェクト ビルド エラー PRJ0046

指定されたコマンドラインが空だったため、コマンドラインを生成できませんでした。

メイクファイルの構成に空のコマンドラインが指定されました。コマンドラインが必要です。
