# <a name="expression-evaluator-error-cxx0038"></a>式エバリュエーター エラー CXX0038

関数の引数の数または型の不一致

指定された関数呼び出しが、関数のプロトタイプと一致しません。

正しい数の引数で呼び出しを再入力してください。 必要に応じて、プロトタイプに一致するように各引数をキャストします。

このエラーは CAN0038 と同じです。
