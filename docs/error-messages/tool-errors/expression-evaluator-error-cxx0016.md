# <a name="expression-evaluator-error-cxx0016"></a>式エバリュエーター エラー CXX0016

定数が大きすぎます。

C 式エバリュエーターでは、4294967295より大きい符号なし整数定数 (0FFFFFFFF 16 進数) または絶対値が約 1.8 E + 308 を超える浮動小数点定数を受け入れることはできません。

このエラーは CAN0016 と同じです。
