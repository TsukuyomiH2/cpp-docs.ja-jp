# <a name="math-error-m6205"></a>数値演算エラー M6205

' function ': _TLOSS エラー

有意性 (有効桁数) の合計が失われました。

このエラーは、sin、cos、または tan のオペランドとして非常に大きい数値を指定することによって発生することがあります。これは、オペランドを0から 2 * pi までの数値に減らす必要があるためです。
