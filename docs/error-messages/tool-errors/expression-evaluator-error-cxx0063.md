# <a name="expression-evaluator-error-cxx0063"></a>式エバリュエーター エラー CXX0063

オーバーロードされた演算子-> サポートされていません

式で、オーバーロードされたクラスメンバーアクセス演算子 ( **->** ) が使用されています。

このエラーは CAN0063 と同じです。
