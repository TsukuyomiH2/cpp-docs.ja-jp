# <a name="expression-evaluator-error-cxx0008"></a>式エバリュエーター エラー CXX0008

**"文字列の末尾に" がありません**

文字列リテラルの終わりに必要な二重引用符が見つかりませんでした。 式を再入力します。リテラル文字列は二重引用符で囲みます。

このエラーは CAN0008 と同じです。
