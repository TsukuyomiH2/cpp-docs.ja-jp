# <a name="resource-compiler-fatal-error-rc1004"></a>リソース コンパイラの致命的なエラー RC1004

予期しないファイルの終わりが見つかりました

このエラーは、テキストファイルの最終行にラインフィードと復帰文字がないことが原因で発生する場合があります。
