# <a name="expression-evaluator-error-cxx0058"></a>式エバリュエーター エラー CXX0058

オーバーロードされた演算子が見つかりません

クラス型が式の左オペランドとして指定されましたが、オーバーロードされた演算子がクラスに対して定義されていませんでした。

このエラーは CAN0058 と同じです。
