# <a name="expression-evaluator-error-cxx0050"></a>式エバリュエーター エラー CXX0050

:: の左側はクラス、構造体、共用体でなければなりません

スコープ解決演算子 (`::`) の左側の記号が、クラス、構造体、または共用体ではありませんでした。

このエラーは CAN0050 と同じです。
