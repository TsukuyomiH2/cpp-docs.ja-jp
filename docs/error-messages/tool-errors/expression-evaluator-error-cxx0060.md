# <a name="expression-evaluator-error-cxx0060"></a>式エバリュエーター エラー CXX0060

レジスタは使用できません

使用できないレジスタが式に指定されています。 このエラーは、を実行しているコンピューターに存在しないレジスタにアクセスしようとしたことが原因で発生する場合があります。

このエラーは CAN0060 と同じです。
