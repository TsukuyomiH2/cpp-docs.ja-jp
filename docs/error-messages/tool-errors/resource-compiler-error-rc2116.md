# <a name="resource-compiler-error-rc2116"></a>リソース コンパイラ エラー RC2116

ID には数値が必要です

**DIALOG**ステートメント内の control ステートメントの `id` フィールドには数値が必要です。 コントロール ID に対して、数字または `#define` ステートメントがあることを確認します。
