# <a name="linker-tools-warning-lnk4065"></a>リンカー ツールの警告 LNK4065

' function ' を順序付けることはできません。無効

指定された関数は、パッケージ化された関数としてコンパイルされませんでした。 [/Gy](../../build/reference/gy-enable-function-level-linking.md)を使用して再コンパイルします。
