# <a name="expression-evaluator-error-cxx0057"></a>式エバリュエーター エラー CXX0057

行番号にコードがありません

指定された行番号に対してコードが生成されませんでした。 ブレークポイントとして使用することはできません。

このエラーは CAN0057 と同じです。
