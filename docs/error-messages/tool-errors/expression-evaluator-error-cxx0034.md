# <a name="expression-evaluator-error-cxx0034"></a>式エバリュエーター エラー CXX0034

**演算子と互換性のない型**

指定されたオペランドの型は、操作には有効ではありません。

たとえば、ポインターに任意の値を乗算することはできません。

場合によっては、演算子と互換性のある型にオペランドをキャストする必要があります。
