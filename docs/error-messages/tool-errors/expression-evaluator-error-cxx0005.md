# <a name="expression-evaluator-error-cxx0005"></a>式エバリュエーター エラー CXX0005

サポートされていない演算子

サポートされていない C 演算子が式で指定されました。 サポートされている C 演算子を使用して、同等の式を記述します。

このエラーは CAN0005 と同じです。
