# <a name="expression-evaluator-error-cxx0029"></a>式エバリュエーター エラー CXX0029

構造体ポインターではありません

メンバー選択演算子 ( **->** ) は、構造体へのポインターではない式に適用されました。

式全体が正しくかっこで囲まれていることを確認するか、「address 式を適切な構造体ポインター型にキャストします」と入力します。

このエラーは CAN0029 と同じです。
