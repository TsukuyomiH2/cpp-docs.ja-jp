# <a name="expression-evaluator-error-cxx0067"></a>式エバリュエーター エラー CXX0067

関数の評価はサポートされていません

式に関数呼び出しが含まれています。 一部の式エバリュエーターでは、関数呼び出しはサポートされていません。

このエラーは CAN0067 と同じです。
