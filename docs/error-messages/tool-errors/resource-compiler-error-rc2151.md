# <a name="resource-compiler-error-rc2151"></a>リソース コンパイラ エラー RC2151

文字列定数を再使用できません

**Stringtable** ステートメントで同じ値を2回使用しています。 重複する10進値と16進値が混在していないことを確認します。

**Stringtable** 内の各 ID は一意である必要があります。 効率を最大にするには、16の倍数で始まる連続する定数を使用します。
