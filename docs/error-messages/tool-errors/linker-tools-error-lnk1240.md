# <a name="linker-tools-error-lnk1240"></a>リンカ ツール エラー LNK1240

IDL コンテンツをコンパイルできませんでした

リンカーは、埋め込み IDL をコンパイルするために MIDL を起動しましたが、問題が発生しました。 MIDL によって指定されたエラーを確認します。
