# <a name="resource-compiler-error-rc2107"></a>リソース コンパイラ エラー RC2107

**数値のコマンド値が必要です**

リソースコンパイラは、**アクセラレータ** ステートメントで数値 *idvalue* フィールドを予期していました。 定数を使用して値を指定していること `#define` と、定数のスペルが正しいことを確認してください。
