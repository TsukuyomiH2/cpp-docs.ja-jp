# <a name="resource-compiler-error-rc2107"></a>リソース コンパイラ エラー RC2107

**数値のコマンド値が必要です**

リソースコンパイラは、**アクセラレータ**ステートメントで数値*idvalue*フィールドを予期していました。 値を指定するために `#define` 定数が使用されていることと、定数のスペルが正しいことを確認してください。
