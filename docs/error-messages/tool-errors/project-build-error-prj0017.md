# <a name="project-build-error-prj0017"></a>プロジェクト ビルド エラー PRJ0017

現在の作業ディレクトリが無効です。

現在の作業ディレクトリへのパスが _MAXPATH を超えています。 このエラーを解決するには、このような深いレベルでプロジェクトを作成しないでください。
