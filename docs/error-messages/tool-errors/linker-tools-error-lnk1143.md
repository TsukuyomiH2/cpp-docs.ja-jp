# <a name="linker-tools-error-lnk1143"></a>リンカ ツール エラー LNK1143

ファイルが無効であるか壊れています: COMDAT セクション番号のシンボルがありません

このエラーは、オブジェクトファイルが壊れている場合に発生する可能性があります。

ファイルをリビルドします。
