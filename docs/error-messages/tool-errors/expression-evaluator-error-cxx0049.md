# <a name="expression-evaluator-error-cxx0049"></a>式エバリュエーター エラー CXX0049

古い OMF-プログラムを再リンクしてください

プログラムは古い OMF (オブジェクトモジュール形式) を使用しています。

現在のリンカーバージョンを使用してプログラムを再リンクします。

このエラーは CAN0049 と同じです。
