# <a name="expression-evaluator-error-cxx0023"></a>式エバリュエーター エラー CXX0023

無効な基数

C 式エバリュエーターでは、指定された基数は認識されません。 有効なのは、10進数、16進数、および8進数の mux のみです。

このエラーは CAN0023 と同じです。
