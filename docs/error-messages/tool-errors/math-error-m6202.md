# <a name="math-error-m6202"></a>数値演算エラー M6202

' function ': _SING エラー

指定された関数の引数が、この関数の特異値でした。 この引数に対して関数が定義されていません。

このエラーは、関数 `_matherr` 名、引数、およびエラーの種類を使用して関数を呼び出します。 関数を書き直して、 `_matherr` 特定の実行時の浮動小数点演算エラーの処理をカスタマイズすることができます。
