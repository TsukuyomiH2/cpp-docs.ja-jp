# <a name="expression-evaluator-error-cxx0044"></a>式エバリュエーター エラー CXX0044

_based (void) ポインターの使用には、> 演算子が必要です

`void` に基づくポインターを直接使用することはできません。 **: >** 演算子を使用して、完全なポインターを形成する必要があります。

このエラーは CAN0044 と同じです。
