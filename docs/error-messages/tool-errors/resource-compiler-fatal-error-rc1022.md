# <a name="resource-compiler-fatal-error-rc1022"></a>リソース コンパイラの致命的なエラー RC1022

' #endif' が必要です。

`#if`、 **#ifdef**、または **#ifndef**ディレクティブが `#endif` ディレクティブで終了しませんでした。

このステートメントの前に、`#if`、 **#ifdef**、または **#ifndef**ステートメントが有効になっていることを確認してください。
