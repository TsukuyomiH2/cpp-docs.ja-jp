# <a name="expression-evaluator-error-cxx0002"></a>式エバリュエーター エラー CXX0002

ユーザーメモリへのアクセスエラー

この式では、デバッグ中のプログラムに割り当てられていないメモリを参照しようとしています。

このエラーは CAN0002 と同じです。
