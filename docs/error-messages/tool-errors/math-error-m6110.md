# <a name="math-error-m6110"></a>数値演算エラー M6110

スタック オーバーフロー

浮動小数点式により、浮動小数点のスタックオーバーフローが発生しました。

スタックオーバーフロー浮動小数点例外は、8087/287/387 コプロセッサによって通常サポートされる8レベルに加えて、7レベルの制限に達するまでトラップされます。

プログラムは終了コード138で終了します。
