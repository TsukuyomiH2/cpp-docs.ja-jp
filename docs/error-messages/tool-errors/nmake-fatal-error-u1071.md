# <a name="nmake-fatal-error-u1071"></a>NMAKE の致命的なエラー U1071

ターゲット ' targetname ' の依存関係ツリーでの循環

指定されたターゲットの依存関係ツリーに循環依存関係が存在します。 指定されたターゲットは、指定されたターゲットの依存のいずれかに依存しています。 循環依存関係が無効です。
