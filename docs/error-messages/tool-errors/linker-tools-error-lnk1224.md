# <a name="linker-tools-error-lnk1224"></a>リンカ ツール エラー LNK1224

イメージのベースアドレスが無効です

イメージに無効なベースアドレスが指定されました。 ベースアドレスは、64 KB でアラインする必要があります (最後の4桁の16進数はゼロでなければなりません)。また、image base は、32ビットの符号付きまたは符号なしの値内に収まる必要があります。
