# <a name="resource-compiler-error-rc2124"></a>リソース コンパイラ エラー RC2124

空のメニューは使用できません

**Menu ステートメントで** メニュー項目が定義される前に、 **END** キーワードが表示されます。 リソースコンパイラでは、空のメニューは許可されません。 **メニュー** ステートメント内に、開いている引用符がないことを確認します。
