# <a name="linker-tools-error-lnk1221"></a>リンカ ツール エラー LNK1221

サブシステムは推論できないため、定義する必要があります

リンカーには、対象とするサブシステムを推測するための十分な情報がありません。

このエラーを修正するには、 [/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)を使用します。
