# <a name="nmake-fatal-error-u1007"></a>NMAKE の致命的なエラー U1007

名前に二重引用符を使用することはできません

指定されたターゲット名またはファイル名に二重引用符 ( **"** ) が含まれています。

二重引用符はファイル名を囲むことができますが、その中に含めることはできません。
