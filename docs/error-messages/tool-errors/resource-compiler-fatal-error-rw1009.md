# <a name="resource-compiler-fatal-error-rw1009"></a>リソース コンパイラの致命的なエラー RW1009

リソース名の作成エラー

指定された .res ファイルを作成できませんでした。 読み取り専用ドライブに作成されていないことを確認します。 ファイルが作成されているかどうかを確認するには、/V を使用します。
