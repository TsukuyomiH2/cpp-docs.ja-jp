# <a name="linker-tools-warning-lnk4199"></a>リンカー ツールの警告 LNK4199

/DELAYLOAD: dllname は無視されます。dllname からインポートが見つかりませんでした

リンカーは、`dllname` エクスポートする関数を必要としないため、`dllname` を無視します。
