# <a name="expression-evaluator-error-cxx0040"></a>式エバリュエーター エラー CXX0040

関数には暗黙的な変換が必要です

C 式エバリュエーターでは、コンストラクター呼び出しに関連する暗黙的な変換はサポートされていません。

このエラーは CAN0040 と同じです。
