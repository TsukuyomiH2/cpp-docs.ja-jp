# <a name="resource-compiler-warning-rw4001"></a>リソース コンパイラの警告 RW4001

.実行可能ファイルの処理オプション (/L/M/P/T/K/E/31 または/30)

EXE 処理オプションが指定されましたが、処理する実行可能ファイルがありませんでした。 これらのオプションは、.res ファイルと共に使用しないでください。
