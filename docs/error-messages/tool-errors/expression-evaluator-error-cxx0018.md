# <a name="expression-evaluator-error-cxx0018"></a>式エバリュエーター エラー CXX0018

レジスタ名が正しくありません

指定されたレジスタが存在しないか、または表示できません。

ウォッチウィンドウには、次のレジスタを表示できます。

AX SP DS IP

BX BP ES FL

CX SI

DX DI CS SS

EAX ESP DS EIP

EBX EBP ES EFL

ECX ESI FS

EDX EDI CS

このエラーは CAN0018 と同じです。
