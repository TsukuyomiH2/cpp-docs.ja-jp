# <a name="linker-tools-error-lnk1248"></a>リンカ ツール エラー LNK1248

> イメージサイズ ('*size*') が最大許容サイズを超えています (8000万)

リンカーは、出力ファイルのサイズが32ビットのプログラムイメージで可能な最大サイズを超えていると判断しました。 プログラムを複数の Dll にすることができます。
