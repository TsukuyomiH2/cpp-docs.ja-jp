# <a name="resource-compiler-error-rc2103"></a>リソース コンパイラ エラー RC2103

文字列リテラルに予期しないファイルの終わりがあります

文字列の末尾の前に、ファイルの終わりが見つかりました。 文字列の終わりに二重引用符 ( **"** ) がありません。
