# <a name="expression-evaluator-error-cxx0045"></a>式エバリュエーター エラー CXX0045

関数ではありません

プログラムのシンボルに引数リストが指定されましたが、関数名ではありません。

## <a name="example"></a>例

```
queue( alpha, beta )
```

`queue` が関数でない場合。

このエラーは CAN0045 と同じです。
