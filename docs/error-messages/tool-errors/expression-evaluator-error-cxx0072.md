# <a name="expression-evaluator-error-cxx0072"></a>式エバリュエーター エラー CXX0072

エラー: 型情報が見つからないか、不明です

.Pch ファイルがリンクされていないか、/Zi. でコンパイルされていないモジュール内の型への参照がコードに含まれています。

このエラーは CAN0067 と同じです。
