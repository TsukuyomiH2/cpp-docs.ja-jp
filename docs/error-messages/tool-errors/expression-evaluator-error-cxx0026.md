# <a name="expression-evaluator-error-cxx0026"></a>式エバリュエーター エラー CXX0026

書式設定文字列が正しくありません

書式文字列が正しく指定されていません。 式の構文を確認します。

このエラーは CAN0026 と同じです。

詳細については、「 [C++ の書式指定子](/visualstudio/debugger/format-specifiers-in-cpp)」を参照してください。
