# <a name="bscmake-error-bk1515"></a>BSCMAKE エラー BK1515

bscfile: 互換性のないバージョンです: 増分更新できません

.Bsc ファイルは、別のバージョンの BSCMAKE で作成されました。 .Bsc ファイルを再作成します。
