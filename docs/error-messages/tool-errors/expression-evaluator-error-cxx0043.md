# <a name="expression-evaluator-error-cxx0043"></a>式エバリュエーター エラー CXX0043

このポインターは、メンバー関数の外で使用されています。

ポインターは、非 **`this`** 静的メンバー関数に対してのみ使用できます。

このエラーは CAN0043 と同じです。
