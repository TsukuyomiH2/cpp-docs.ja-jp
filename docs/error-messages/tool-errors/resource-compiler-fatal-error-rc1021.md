# <a name="resource-compiler-fatal-error-rc1021"></a>リソース コンパイラの致命的なエラー RC1021

プリプロセッサ コマンド 'string' が無効です

シャープ記号 ( **#** ) の後の文字は、有効なプリプロセッサディレクティブを形成しませんでした。

シャープ記号を識別子の最初の文字として使用することはできません。
