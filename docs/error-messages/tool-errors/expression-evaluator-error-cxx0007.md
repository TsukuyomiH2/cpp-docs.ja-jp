# <a name="expression-evaluator-error-cxx0007"></a>式エバリュエーター エラー CXX0007

右かっこがありません

式に、不均衡かっこが見つかりました。 均衡かっこを使用して式を再入力します。

このエラーは CAN0007 と同じです。
