# <a name="expression-evaluator-error-cxx0028"></a>式エバリュエーター エラー CXX0028

struct/union 要素ではありません

' Struct. Member ' または ' pStruct-> Member ' の形式の式が指定されましたが、\<メンバー > が構造体の要素ではありません。

式が正しく指定されていない可能性があります。

このエラーは CAN0028 と同じです。
