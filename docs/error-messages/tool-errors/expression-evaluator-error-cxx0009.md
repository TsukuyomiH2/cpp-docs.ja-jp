# <a name="expression-evaluator-error-cxx0009"></a>式エバリュエーター エラー CXX0009

' after character 定数がありません。

文字定数の末尾に必要な単一引用符が見つかりませんでした。 文字定数を単一引用符で囲んで、式を再入力します。

このエラーは CAN0009 と同じです。
