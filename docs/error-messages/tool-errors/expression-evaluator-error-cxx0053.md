# <a name="expression-evaluator-error-cxx0053"></a>式エバリュエーター エラー CXX0053

ブレークポイントのバインド中に関数以外のシンボルが一致しました

ブレークポイントとして使用されているシンボルが関数ではありませんでした。 データメンバーをブレークポイントとして指定すると、このエラーが発生する可能性があります。

このエラーは CAN0053 と同じです。
