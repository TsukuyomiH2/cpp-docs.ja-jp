# <a name="expression-evaluator-error-cxx0062"></a>式エバリュエーター エラー CXX0062

コンストラクター呼び出しはサポートされていません

コンストラクターを呼び出した式。 式では、コンストラクターへの明示的な呼び出しや、コンストラクターの呼び出しを必要とする変換を行うことはできません。

このエラーは CAN0062 と同じです。
