# <a name="compiler-error-c2222"></a>コンパイラ エラー C2222

不要な型 'type': 基底クラスまたはメンバーが必要です

初期化子リストで初期化できるのは、1 つの型の基底クラスまたはメンバーのみです。 このエラーを修正するには、初期化子リストでその型の基底クラスまたはメンバーのみが初期化されていることを確認します。
