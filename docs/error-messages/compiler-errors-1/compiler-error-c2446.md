# <a name="compiler-error-c2446"></a>コンパイラ エラー C2446

' operator ': ' type1 ' から ' type1 ' への変換はできません。

コンパイラは `type1` を `type2`に変換できません。 変換は、C/C++セマンティクスに違反しているため意味を持たない場合があります。
