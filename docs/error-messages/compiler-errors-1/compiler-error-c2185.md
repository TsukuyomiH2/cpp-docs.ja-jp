# <a name="compiler-error-c2185"></a>コンパイラ エラー C2185

' identifier ': ベース割り当てが無効です

レジスタ変数または自動 (ローカル) 変数が `__based`として宣言されています。 `__based`として宣言できるのは、グローバル変数だけです。
