# <a name="compiler-error-c2307"></a>コンパイラ エラー C2307

インクリメンタルコンパイルが有効になっている場合、プラグマ ' pragma ' は外部関数でなければなりません

`data_seg`インクリメンタルコンパイルを使用している場合は、関数間にプラグマを配置する必要があります。
