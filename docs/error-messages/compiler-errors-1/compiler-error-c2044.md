# <a name="compiler-error-c2044"></a>コンパイラ エラー C2044

'continue' が正しくありません。

[Continue](../../cpp/continue-statement-cpp.md)は **`do`** 、、 **`for`** 、またはステートメント内でのみ有効です **`while`** 。
