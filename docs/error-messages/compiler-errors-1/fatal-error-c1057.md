# <a name="fatal-error-c1057"></a>致命的なエラー C1057

マクロ展開中に予期せぬ EOF を検出しました。

コンパイラはマクロ呼び出しの引数を収集しているときに、ソースファイルの最後に到達しました。マクロ呼び出しに右かっこがないことが原因である可能性があります。
