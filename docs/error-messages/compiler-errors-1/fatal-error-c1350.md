# <a name="fatal-error-c1350"></a>致命的なエラー C1350

dll ' dll ' の読み込み中にエラーが発生しました: dll が見つかりません

試行された操作をサポートする DLL が見つかりませんでした。

これは、インストールに問題があることを示しており、製品 CD から DLL を再インストールする必要があります。
