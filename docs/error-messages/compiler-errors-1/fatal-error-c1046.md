# <a name="fatal-error-c1046"></a>致命的なエラー C1046

コンパイラの制限: 構造の入れ子のレベルが深すぎます。

構造体、共用体、またはクラスが、入れ子の制限である15レベルを超えています。 入れ子レベルを減らすために、定義を書き直してください。 を使用して **`typedef`** 1 つ以上の入れ子構造体を定義することで、構造体、共用体、またはクラスを複数の部分に分割します。
