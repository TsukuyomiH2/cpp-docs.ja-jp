# <a name="compiler-error-c2426"></a>コンパイラ エラー C2426

' token ': ' context ' に無効な演算子があります。

このコンテキストでは、トークンを演算子として使用することはできません。 たとえば、インデックス演算子を入れ子にすることはできません。
