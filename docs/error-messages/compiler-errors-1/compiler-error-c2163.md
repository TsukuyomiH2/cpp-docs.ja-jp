# <a name="compiler-error-c2163"></a>コンパイラ エラー C2163

' function ': 組み込み関数として使用できません。

`intrinsic`または `function` プラグマに、組み込み形式で使用できない関数が一覧表示されています。 たとえば、/clr プログラミングを使用するプログラムをコンパイルするときに、特定の組み込みを使用することはできません。
