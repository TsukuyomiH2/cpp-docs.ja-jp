# <a name="compiler-error-c2163"></a>コンパイラ エラー C2163

' function ': 組み込み関数として使用できません。

`intrinsic` または `function` プラグマは、組み込み形式で使用できない関数を一覧表示します。 たとえば、/clr プログラミングを使用するプログラムをコンパイルするときに、特定の組み込みを使用することはできません。
