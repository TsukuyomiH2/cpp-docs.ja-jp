# <a name="fatal-error-c1053"></a>致命的なエラー C1053

'\<identifier > ': 関数が大きすぎます

コンパイルするには関数が大きすぎます。

### <a name="to-fix-by-using-the-following-possible-solutions"></a>修復の可能性がある解決策

1. 最適化せずにコンパイルしてみてください。

1. 大きな関数を小さな関数に分割します。

1. インライン関数の呼び出しを減らします。
