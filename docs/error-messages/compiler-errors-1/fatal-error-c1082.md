# <a name="fatal-error-c1082"></a>致命的なエラー C1082

filetype ファイルを閉じることができません。'file': メッセージ

"ファイル番号が正しくありません" というメッセージが表示される場合は、ファイルがバックグラウンドでのコンパイル中にフォアグラウンドで閉じられている可能性があります。
