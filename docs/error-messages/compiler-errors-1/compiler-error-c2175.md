# <a name="compiler-error-c2175"></a>コンパイラ エラー C2175

' locale ': ロケールが無効です

指定されたロケールが無効です。 サポートされているロケールについては、「*ランタイムライブラリリファレンス*」の「[言語および国/地域識別文字列](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)」を参照してください。
