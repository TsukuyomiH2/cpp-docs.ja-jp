# <a name="compiler-error-c2018"></a>コンパイラエラー C2018

文字 'hexnumber' は認識できません

ソースファイルに予期しない ASCII 文字が含まれています。これは、16進数で識別されます。 このエラーを解決するには、文字を削除します。
