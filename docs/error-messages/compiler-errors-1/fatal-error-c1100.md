# <a name="fatal-error-c1100"></a>致命的なエラー C1100

OLE を初期化できません: システムエラーメッセージ

コンパイラは、コンポーネントオブジェクトモデル (COM) ライブラリを初期化できません。 「 [CoInitialize](/windows/win32/api/objbase/nf-objbase-coinitialize)」を参照してください。
