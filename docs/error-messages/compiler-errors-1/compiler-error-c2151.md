# <a name="compiler-error-c2151"></a>コンパイラ エラー C2151

複数の言語属性

関数には **`__cdecl`** 、 **`__stdcall`** **`__fastcall`** 呼び出し規約を指定する複数のキーワード (、、または) があります。
