# <a name="fatal-error-c1009"></a>致命的なエラー C1009

コンパイラの制限: マクロの入れ子のレベルが深すぎます。

コンパイラが同時に拡張しようとしているマクロの数が多すぎます。 コンパイラでは、入れ子になったマクロの256レベルの制限があります。 入れ子になったマクロをより単純なマクロに分割します。
