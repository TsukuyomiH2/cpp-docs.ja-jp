# <a name="compiler-error-c2182"></a>コンパイラ エラー C2182

'identifier': 'void' 型の使用法が正しくありません

変数が型として宣言されて **`void`** います。

次の例では C2182 が生成されます。

```cpp
// C2182.cpp
// compile with: /c
int main() {
   int i = 10;
   void &ir = i;   // C2182 cannot have a reference to type void
   int &ir = i;   // OK
}
```
