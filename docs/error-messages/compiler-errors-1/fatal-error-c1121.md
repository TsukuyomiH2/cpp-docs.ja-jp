# <a name="fatal-error-c1121"></a>致命的なエラー C1121

CryptoAPI への呼び出しに失敗しました

コンパイラが CryptoAPI を呼び出しましたが、呼び出しに失敗しました。 Visual Studio を再インストールし、場合によってはオペレーティングシステムを再インストールします。
