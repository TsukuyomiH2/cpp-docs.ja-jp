# <a name="compiler-error-c2320"></a>コンパイラ エラー C2320

> アクセス指定子 '*指定子*' の後には ': ' が必要です

キーワード **`public`** 、 **`protected`** 、またはの **`private`** 後にはコロンが必要です。
