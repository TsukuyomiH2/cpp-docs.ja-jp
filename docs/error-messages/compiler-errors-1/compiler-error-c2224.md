# <a name="compiler-error-c2224"></a>コンパイラ エラー C2224

' identifier ' の左側は構造体型または共用体型でなければなりません

ピリオド (.) の左側のオペランドが、クラス、構造体、または共用体ではありません。

このエラーは、未定義の変数 (したがって、型) である左オペランドによって発生することがあり **`int`** ます。
