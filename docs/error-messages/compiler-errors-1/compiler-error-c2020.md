# <a name="compiler-error-c2020"></a>コンパイラエラー C2020

' member ': ' class ' メンバーの再定義

基底クラスまたは構造体から継承されたメンバーが再定義されます。 継承されたメンバーは、基底クラスの `virtual` として宣言されている場合を除き、再定義できません。
