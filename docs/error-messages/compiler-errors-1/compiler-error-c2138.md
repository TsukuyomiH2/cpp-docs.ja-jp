# <a name="compiler-error-c2138"></a>コンパイラ エラー C2138

メンバーを含まない列挙型を定義することは無効です

/Za (Microsoft 拡張機能を無効にする) が選択されている場合、列挙体には少なくとも1つのメンバーが必要です。
