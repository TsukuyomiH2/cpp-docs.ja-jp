# <a name="compiler-error-c2077"></a>コンパイラエラー C2077

非スカラーフィールド初期化子 ' identifier '

非スカラー (構造体、共用体、配列、またはクラス) を使用してビットフィールドを初期化しようとしました。 整数または浮動小数点値を使用します。
