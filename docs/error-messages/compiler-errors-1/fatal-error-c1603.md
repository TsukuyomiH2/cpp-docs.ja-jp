# <a name="fatal-error-c1603"></a>致命的なエラー C1603

インラインアセンブリブランチターゲットは ' number ' バイトで範囲外です

JCXZ または JECXZ 命令と、指定されたターゲットラベル間の計算された距離が128バイトを超えています。 コードを更新して、ラベルが命令の近くになるようにします。
