# <a name="fatal-error-c1045"></a>致命的なエラー C1045

コンパイラの制限 : 外部参照の入れ子のレベルがコンパイラの限界を超えています。

入れ子になった外部参照がコンパイラの制限を超えています。 入れ子になった外部参照は、 `extern` "C++" などの外部リンケージの種類で許可されます。 エラーを解決するには、入れ子になった外部参照の数を減らします。
