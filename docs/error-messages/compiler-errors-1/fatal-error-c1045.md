# <a name="fatal-error-c1045"></a>致命的なエラー C1045

コンパイラの制限 : 外部参照の入れ子のレベルがコンパイラの限界を超えています。

入れ子になった外部参照がコンパイラの制限を超えています。 入れ子になった外部リンケージは、"C++" などの外部リンケージ型で使用でき **`extern`** ます。 エラーを解決するには、入れ子になった外部参照の数を減らします。
