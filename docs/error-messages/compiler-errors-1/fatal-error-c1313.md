# <a name="fatal-error-c1313"></a>致命的なエラー C1313

コンパイラの制限: 型ブロックは、数値レベルよりも深く入れ子にすることはできません

例外処理 (または構造化例外処理) ブロックの入れ子のレベルが深すぎます。  コードを簡略化します。
