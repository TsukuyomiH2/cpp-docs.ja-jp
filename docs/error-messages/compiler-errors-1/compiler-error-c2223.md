# <a name="compiler-error-c2223"></a>コンパイラ エラー C2223

'-> identifier ' の左側は構造体または共用体を指す必要があります

`->` の左側のオペランドがクラス、構造体、または共用体へのポインターではありません。

このエラーは、未定義の変数 (したがって、型 `int`) である左オペランドによって発生することがあります。
