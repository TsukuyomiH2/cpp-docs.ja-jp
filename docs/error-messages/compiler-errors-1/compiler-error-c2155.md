# <a name="compiler-error-c2155"></a>コンパイラ エラー C2155

'? ': 左オペランドが無効です。算術型またはポインター型が必要です。

`?` の左側の式を0と比較することはできません。 0と比較できる算術式またはポインター式を使用する必要があります。
