# <a name="fatal-error-c1108"></a>致命的なエラー C1108

> DLL が見つかりません: '*dll name*'

指定された DLL (*dll 名*) がパスに見つかりませんでした。

このエラーを解決するには、Visual C++ を再インストールするか、適切な .dll ファイルをインストールされているコンピューターにコピーします。
