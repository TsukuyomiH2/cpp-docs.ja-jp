# <a name="compiler-error-c2122"></a>コンパイラ エラー C2122

' identifier ': 名前リストのプロトタイプパラメーターが正しくありません

パラメーターが有効な型ではありません。 ANSI C は、ユーザー定義型をサポートしていません。
