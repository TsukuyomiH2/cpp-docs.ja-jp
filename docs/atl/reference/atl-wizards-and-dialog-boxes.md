# <a name="atl-wizards-and-dialog-boxes"></a>ATL ウィザードとダイアログボックス

Active Template Library (ATL) ウィザードでは、さまざまな種類の COM オブジェクトの定型コードが生成されます。 ウィザードを実行するには、 **ソリューションエクスプローラー** でプロジェクトのショートカットメニューを開き、[ **追加**]、[ **クラス**] の順に選択します。
