# <a name="iatlautothreadmodule-class"></a>IAtlAutoThreadModule クラス

このクラスは、メソッドへのインターフェイスを表し `CreateInstance` ます。

> [!IMPORTANT]
> このクラスとそのメンバーは、Windows ランタイムで実行されるアプリケーションでは使用できません。

```
__interface IAtlAutoThreadModule
```
