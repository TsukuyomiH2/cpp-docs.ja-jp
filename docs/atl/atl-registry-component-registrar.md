# <a name="atl-registry-component-registrar"></a>ATL レジストリ コンポーネント (レジストラー)

ATL レジストラーは、カスタムインターフェイスを使用してシステムレジストリへのアクセスを最適化します。 レジストラーはフリースレッドであり、C++ クライアントのコードを静的にリンクすることができます。

> [!NOTE]
> ATL レジストラーのソースコードは atlmfc\include\atliface.h. にあります。
