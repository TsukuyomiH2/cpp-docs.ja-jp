# <a name="persistence-classes"></a>永続化クラス

次のクラスは、オブジェクトの永続性を実装します。

- [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md) オブジェクトのプロパティを読み込んで、プロパティバッグに保存することをクライアントに許可します。

- [IPersistStreamInitImpl](../atl/reference/ipersiststreaminitimpl-class.md) クライアントがオブジェクトの永続データを読み込んでストリームに保存できるようにします。

- [IPersistStorageImpl](../atl/reference/ipersiststorageimpl-class.md) オブジェクトの永続データをクライアントが読み込み、ストレージに保存できるようにします。
