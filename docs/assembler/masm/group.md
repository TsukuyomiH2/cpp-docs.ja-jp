# <a name="group"></a>GROUP

(32 ビット MASM のみ。)指定された *セグメント* を *名前* と呼ばれるグループに追加します。

## <a name="syntax"></a>構文

> *名前***グループ***セグメント*⟦__、__ *セグメント*...⟧
