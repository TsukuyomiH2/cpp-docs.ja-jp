# <a name="operator-lroffset"></a>LROFFSET 演算子

*式* のオフセットを返します。 **オフセット** と同じですが、ローダーによって解決されるオフセットが生成されます。これにより、Windows がコードセグメントを再配置できるようになります。

## <a name="syntax"></a>構文

> **Lroffset** *式*
