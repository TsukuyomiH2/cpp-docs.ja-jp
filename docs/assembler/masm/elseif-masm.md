# <a name="elseif"></a>ELSEIF

[ELSE](else-masm.md)と[if](if-masm.md)を1つのステートメントに結合します。

## <a name="syntax"></a>構文

> **ELSEIF** *constantExpression*\
> *命令*\
> ⟦**ELSE**\
> *else-ステートメント*⟧ \
> **ENDIF**
