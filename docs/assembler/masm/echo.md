# <a name="echo"></a>ECHO

標準出力デバイス (既定では画面) に *メッセージ* を表示します。

## <a name="syntax"></a>構文

> **エコー** *メッセージ*

## <a name="remarks"></a>解説

[% OUT](percent-out.md)と同じです。
