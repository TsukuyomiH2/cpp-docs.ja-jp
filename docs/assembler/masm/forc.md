# <a name="forc"></a>FORC

*文字列* 内の各文字に対して1回繰り返されるブロックをマークします。各繰り返しの *現在の文字を置換し* ます。

## <a name="syntax"></a>構文

> **Forc** *パラメーター*、 __\<__*string*__>__\
> *命令*\
> **ENDM**

## <a name="remarks"></a>解説

[Irpc](irpc.md)と同じです。
