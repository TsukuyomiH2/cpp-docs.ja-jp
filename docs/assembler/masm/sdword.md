# <a name="sdword"></a>SDWORD

各 *初期化子* について、署名された2単語 (4 バイト) のストレージを割り当て、必要に応じて初期化します。 型が有効である任意の場所で、型指定子として使用することもできます。

## <a name="syntax"></a>構文

> *名前* **sdword** *初期化子* ⟦__、__ *初期化子* ...⟧
