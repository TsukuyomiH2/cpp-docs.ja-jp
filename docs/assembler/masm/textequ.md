# <a name="textequ"></a>TEXTEQU

*名前* に *textitem* を割り当てます。 *Textitem* には、リテラル文字列、で始まる定数、 `%` またはマクロ関数によって返される文字列を指定できます。

## <a name="syntax"></a>構文

> *名前* **TEXTEQU** ⟦*textitem*⟧
