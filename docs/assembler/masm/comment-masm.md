# <a name="comment"></a>COMMENT

区切り記号と同じ行にあるまたはとの間のすべての *テキスト* をコメントとして扱います。

## <a name="syntax"></a>構文

> **コメント***区切り記号*⟦*text*⟧ \
> ⟦*text*⟧ \
> ⟦*text*⟧ *delimiter* ⟦*text*⟧
