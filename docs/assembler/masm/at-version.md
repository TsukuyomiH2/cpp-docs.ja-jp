# <a name="version"></a>\@バージョン

コマンドラインで1つの数値 (テキストマクロ) として報告された MASM のメジャーバージョンとマイナーバージョン。 たとえば、MASM version 14.23.28107.0 では、"1423" が生成されます。

## <a name="syntax"></a>構文

> **\@Version**
