# <a name="elseif2"></a>ELSEIF2

**OPTION: SETIF2** が **TRUE** の場合、すべてのアセンブリパスで [ELSEIF](elseif-masm.md)ブロックが評価されます。

## <a name="syntax"></a>構文

> **ELSEIF2**

## <a name="remarks"></a>解説

MASM 5.1 vs MASM 6.1 の2パス動作の詳細については、「 [IF1 AND IF2](if2.md)」を参照してください。
