# <a name="ml-nonfatal-error-a2044"></a>ML の致命的でないエラー A2044

**ファイルに無効な文字があります**

このソースファイルには、演算子またはその他の有効な文字として認識されなかったコメント、文字列、またはリテラル以外の文字が含まれていました。
