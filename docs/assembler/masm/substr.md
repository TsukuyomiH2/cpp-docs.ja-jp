# <a name="substr"></a>SUBSTR

*位置* から始まる *textitem* の部分文字列を返します。 *Textitem* には、リテラル文字列、で始まる定数、 `%` またはマクロ関数によって返される文字列を指定できます。

## <a name="syntax"></a>構文

> *名前* **SUBSTR** *textitem*__、__ *position* ⟦__、__ *length*⟧
