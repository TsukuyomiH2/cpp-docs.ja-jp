# <a name="union"></a>UNION

1つ以上のデータ型の和集合を宣言します。 *フィールド宣言* は、有効なデータ定義である必要があります。 入れ子になった **共用体** の定義で [end](ends-masm.md) *name* ラベルを省略します。

## <a name="syntax"></a>構文

> *name* **UNION** ⟦*alignment*⟧⟦__,__ **一意** でない⟧ \
> *フィールド宣言*\
> ⟦*name*⟧ **end**
