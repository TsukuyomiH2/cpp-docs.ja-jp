# <a name="real4"></a>REAL4

各 *初期化子* に対して、単精度 (4 バイト) の浮動小数点数を割り当てて、必要に応じて初期化します。

## <a name="syntax"></a>構文

> *名前*  **REAL4** *initializer* ⟦__、__ *初期化子* ...⟧
