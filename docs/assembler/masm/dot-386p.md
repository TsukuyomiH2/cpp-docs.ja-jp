# <a name="386p-32-bit-masm"></a>386P (32 ビット MASM)

80386プロセッサのすべての命令 (特権を含む) のアセンブリを有効にします。以降のプロセッサで導入された命令のアセンブリを無効にします。 (32 ビット MASM のみ。)

## <a name="syntax"></a>構文

> **.386P**

## <a name="remarks"></a>解説

では、80387命令も有効になっています。
