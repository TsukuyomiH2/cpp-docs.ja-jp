# <a name="struct"></a>STRUCT

指定された*フィールド宣言*を持つ構造体型を宣言します。 各フィールドは、有効なデータ定義である必要があります。 [STRUC](struc.md)と同じです。

## <a name="syntax"></a>構文

> *name* **STRUCT** ⟦*alignment*⟧⟦__,__ **一意**でない⟧ \
> *フィールド宣言*\
> *名前*の**終わり**

## <a name="remarks"></a>解説

*名前*引数は、開始および終了ステートメントと同じである必要があります。
