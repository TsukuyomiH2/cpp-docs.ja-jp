# <a name="mmx-32-bit-masm"></a>.MMX (32 ビット MASM)

MMX または単一命令、複数データ (SIMD) 命令のアセンブリを有効にします。 (32 ビット MASM のみ。)

## <a name="syntax"></a>構文

> **.MMX**
