# <a name="instr"></a>\@InStr

String1 内の *string2* の最初の出現 *箇所を検索* するマクロ関数。 *string1* 内の *位置* から始まります。 *位置* が表示されない場合は、 *string1* の先頭から検索が開始されます。 位置の整数を返します。 *string2* が見つからない場合は0を返します。

## <a name="syntax"></a>構文

> **\@ InStr (** ⟦*position*⟧__,__ *string1*__,__ *string2* **)**
