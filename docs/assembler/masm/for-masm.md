# <a name="for"></a>FOR

各 *引数* に対して1回繰り返されるブロックをマークします。各繰り返しでは、現在の *引数* が *パラメーター* を置換します。

## <a name="syntax"></a>構文

>  *パラメーター* ⟦**: REQ**  |  __: =__*default*⟧ **、 \<** _argument_ ⟦__,__ *argument* ...⟧**>**\
> *命令*\
> **ENDM**

## <a name="remarks"></a>解説

[IRP](irp.md)と同じです。
