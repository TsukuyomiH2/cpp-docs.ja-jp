# <a name="db"></a>DB

各 *初期化子* のストレージのバイトを割り当て、必要に応じて初期化します。 **DB** は [BYTE](byte-masm.md)のシノニムです。

## <a name="syntax"></a>構文

> ⟦*name*⟧ **DB** *初期化子* ⟦__、__ *初期化子* ...⟧

## <a name="remarks"></a>解説

型が有効である任意の場所で、型指定子として使用することもできます。
