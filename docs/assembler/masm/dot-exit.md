# <a name="exit-32-bit-masm"></a>.EXIT (32 ビット MASM)

終了コードを生成します。 (32 ビット MASM のみ。)

## <a name="syntax"></a>構文

> **.** ⟦*式*⟧を終了します。

## <a name="remarks"></a>解説

シェルにオプションの *式* を返します。
