# <a name="sword"></a>SWORD

各 *初期化子* について、署名された単語 (2 バイト) を割り当てて、必要に応じて初期化します。 型が有効である任意の場所で、型指定子として使用することもできます。

## <a name="syntax"></a>構文

> *名前***ソード***初期化子*⟦__、__ *初期化子*...⟧
