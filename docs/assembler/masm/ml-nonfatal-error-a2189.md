# <a name="ml-nonfatal-error-a2189"></a>ML の致命的でないエラー A2189

**セグメントアラインメントとの組み合わせが無効です**

**ALIGN** またはディレクティブによって指定されたアラインメント `EVEN` が、 **segment** ディレクティブで指定された現在のセグメントのアラインメントを超えています。
