# <a name="typedef"></a>TYPEDEF

*Type* または *protoDefinition* と同じ *名前* の新しい型または **PROTO** を定義します。

## <a name="syntax"></a>構文

>  **TYPEDEF** *型*  |  **PROTO** *protoDefinition* に名前を指定します
