# <a name="datasize"></a>\@DataSize

小さい、**小さい**、**中**、および **フラット** モデルの場合は0、**コンパクト** モデルと **大規模** モデルの場合は1、**大きな** モデルの場合は 2 (数値の等価)。 

## <a name="syntax"></a>構文

> **\@DataSize**
