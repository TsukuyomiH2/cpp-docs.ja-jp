# <a name="ymmword"></a>YMMWORD

Intel Advanced Vector Extensions (AVX) 命令を含む256ビットマルチメディアオペランドで使用されます。

## <a name="syntax"></a>構文

> **YMMWORD**

## <a name="remarks"></a>解説

**Ymmword** は、AVX 組み込みの場合と同じ型を表すことを目的としてい `__m256` ます。
