# <a name="ifdif-ifdifi"></a>IFDIF、IFDIFI

テキスト項目が異なる場合にアセンブリを許可します。

## <a name="syntax"></a>構文

> **IFDIF** *textitem1*__、__ *textitem2*\
> **Ifdifi** *textitem1*__、__ *textitem2*

## <a name="remarks"></a>解説

**Ifdifi** が指定されている場合、比較では大文字と小文字が区別されません。 完全な構文について [は](if-masm.md) 、「」を参照してください。
