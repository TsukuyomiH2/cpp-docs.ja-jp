# <a name="686p-32-bit-masm"></a>. 686P (32 ビット MASM)

Pentium Pro プロセッサのすべての命令 (特権を含む) のアセンブリを有効にします。 (32 ビット MASM のみ。)

## <a name="syntax"></a>構文

> **.686P**
