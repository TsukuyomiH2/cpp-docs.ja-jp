# <a name="386-32-bit-masm"></a>. 386 (32 ビット MASM)

80386プロセッサの特権のない命令のアセンブリを有効にします。以降のプロセッサで導入された命令のアセンブリを無効にします。 (32 ビット MASM のみ。)

## <a name="syntax"></a>構文

> **. 386**

## <a name="remarks"></a>解説

では、80387命令も有効になっています。
