# <a name="platformdetailsiprintable-interface"></a>Platform::Details::IPrintable インターフェイス

このインターフェイスは内部使用のみを意図しており、開発で使用するためのものではありません。

## <a name="syntax"></a>構文

```cpp
interface class IPrintable;
```

### <a name="remarks"></a>解説

ブラウザーで検査できるため、このインターフェイスは完全を期すためだけに提供されています。
