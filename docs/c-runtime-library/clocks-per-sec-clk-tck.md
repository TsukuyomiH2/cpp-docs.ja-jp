# <a name="clocks_per_sec-clk_tck"></a>CLOCKS_PER_SEC、CLK_TCK

## <a name="syntax"></a>構文

```
#include <time.h>
```

## <a name="remarks"></a>解説

`clock` 関数の戻り値であり、`CLOCKS_PER_SEC` で除算された秒単位の時間を表します。 `CLK_TCK` も同じものですが、古い形式とされます。
