# <a name="_getmbcp"></a>_getmbcp

現在のコード ページを取得します。

## <a name="syntax"></a>構文

```C
int _getmbcp( void );
```

## <a name="return-value"></a>戻り値

現在のマルチバイト コード ページを返します。 戻り値 0 は、シングル バイトのコード ページが使用中であることを示します。

## <a name="remarks"></a>解説

既定では、この関数のグローバル状態はアプリケーションにスコープされます。 これを変更するには[、CRT のグローバル状態を](../global-state.md)参照してください。

|**_getmbcp**|\<mbctype.h>|
