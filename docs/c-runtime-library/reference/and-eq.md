# <a name="and_eq"></a>and_eq

&= 演算子の代替手段。

## <a name="syntax"></a>構文

```C
#define and_eq &=
```

## <a name="remarks"></a>解説

マクロにより &= 演算子が生成されます。

## <a name="example"></a>例

```cpp
// iso646_and_eq.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 3, b = 2, result;

   result= a &= b;
   cout << result << endl;

   result= a and_eq b;
   cout << result << endl;
}
```

```Output
2
2
```
