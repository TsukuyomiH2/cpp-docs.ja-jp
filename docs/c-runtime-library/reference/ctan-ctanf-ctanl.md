# <a name="ctan-ctanf-ctanl"></a>ctan、ctanf、ctanl

複素数のタンジェントを取得します。

## <a name="syntax"></a>構文

```C
_Dcomplex ctan(
   _Dcomplex z
);
_Fcomplex ctan(
   _Fcomplex z
);  // C++ only
_Lcomplex ctan(
   _Lcomplex z
);  // C++ only
_Fcomplex ctanf(
   _Fcomplex z
);
_Lcomplex ctanl(
   _Lcomplex z
);
```

### <a name="parameters"></a>パラメーター

*方向*\
角度をラジアンで表す複素数。

## <a name="return-value"></a>戻り値

*Z*のタンジェント。

|入力|SEH 例外|**_matherr** 例外的|
|-----------|-------------------|--------------------------|
|± ∞、QNAN、IND|なし|_DOMAIN|
|±∞ (**tan**, **tanf**)|INVALID|_DOMAIN|

## <a name="remarks"></a>解説

C++ ではオーバーロードが可能であるため、 **_Fcomplex**と **_Lcomplex**の値を受け取って返す**ctan**のオーバーロードを呼び出すことができます。 C プログラムでは、 **ctan** は常に **_Dcomplex** 値を取得して返します。

## <a name="requirements"></a>必要条件

|ルーチンによって返される値|C ヘッダー|C++ ヘッダー|
|-------------|--------------|------------------|
|**ctan**、  **ctanf**、 **ctanl**|\<complex.h>|\<ccomplex>|

互換性について詳しくは、「 [Compatibility](../../c-runtime-library/compatibility.md)」をご覧ください。
