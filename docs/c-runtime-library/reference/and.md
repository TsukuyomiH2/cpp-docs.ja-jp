# <a name="and"></a>and

&& 演算子の代替手段。

## <a name="syntax"></a>構文

```C

#define and &&
```

## <a name="remarks"></a>解説

マクロにより && 演算子が生成されます。

## <a name="example"></a>例

```cpp
// iso646_and.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   bool a = true, b = false, result;

   boolalpha(cout);

   result= a && b;
   cout << result << endl;

   result= a and b;
   cout << result << endl;
}
```

```Output
false
false
```

## <a name="requirements"></a>必要条件

**ヘッダー:** \<iso646 >
