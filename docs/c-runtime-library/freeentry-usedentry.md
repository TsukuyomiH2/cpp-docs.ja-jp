# <a name="_freeentry-_usedentry"></a>_FREEENTRY、_USEDENTRY

## <a name="syntax"></a>構文

```
#include <malloc.h>
```

## <a name="remarks"></a>解説

これらの定数は、`_heapwalk` ルーチンから **_HEAPINFO** 構造体の **_useflag** 要素に割り当てられた値を表します。 これらはヒープ エントリの状態を示します。
