# <a name="mb_cur_max"></a>MB_CUR_MAX

現在のロケールのマルチバイト文字の最大バイト数を示すマクロ。

## <a name="syntax"></a>構文

```
#include <stdlib.h>
```

## <a name="remarks"></a>解説

コンテキスト: ANSI マルチバイト文字およびワイド文字の変換関数

`MB_CUR_MAX` の値は、現在のロケールのマルチバイト文字の最大バイト数です。
