# <a name="fopen_max-_sys_open"></a>FOPEN_MAX、_SYS_OPEN

## <a name="syntax"></a>構文

```
#include <stdio.h>
```

## <a name="remarks"></a>解説

これは、同時に開くことのできるファイルの最大数です。 `FOPEN_MAX` は、ANSI との互換性がある名前です。 `_SYS_OPEN` は、既存のコードとの互換性のために提供されています。
