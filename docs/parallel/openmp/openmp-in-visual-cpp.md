# <a name="openmp-in-visual-c"></a>Visual C++ の OpenMP

OpenMP C および C++ アプリケーションプログラムインターフェイスを使用すると、複数のプロセッサを効果的に使用するアプリケーションを作成できます。 Visual C++ では、OpenMP 2.0 標準がサポートされています。
