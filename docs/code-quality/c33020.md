# <a name="c33020"></a>C33020

> 警告 C33020: 間違った HRESULT 使用率が検出されました。

これは、HRESULT を返す関数が FALSE または false を返すことを示す、信頼性の高い警告です。

## <a name="example"></a>例

```cpp
#include <Windows.h>

HRESULT foo()
{
    // ......
    return FALSE; // C33020
}
```

これらの警告は、適切な HRESULT 値を使用して修正されます。
```cpp
#include <Windows.h>

HRESULT foo()
{
    // ......
    return E_FAIL; // OK
}
```
