# <a name="c26492-no_const_cast"></a>C26492 NO_CONST_CAST

`const_cast` を使用して `const`を離すことは避けてください。 「 [ C++コアガイドラインの種類 3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type)」を参照してください。
