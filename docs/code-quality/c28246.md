# <a name="c28246"></a>C28246

> 警告 C28246: 関数 ' \<name> '-パラメーター ' ' の注釈が \<parameter> パラメーターの型と一致しません

注釈を作成するときに、__deref 演算子が非ポインター型に適用されました。
