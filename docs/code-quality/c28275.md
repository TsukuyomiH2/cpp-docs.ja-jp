# <a name="c28275"></a>C28275

> 警告 C28275: \_マクロ\_値\_ のパラメーターが null です

この警告は、分析されているコードではなく、モデルファイルに内部エラーがあることを示します。 パラメーターを指定せずに*マクロ値*関数が呼び出されました。
