# <a name="c26460-use_const_reference_arguments"></a>C26460 USE_CONST_REFERENCE_ARGUMENTS

  関数 '% function% ' の参照引数 '% argument% ' を `const`としてマークできます。 「 [ C++コアガイドライン con. 3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref)」を参照してください。
