# <a name="c28204"></a>C28204

> 警告 C28204: \<関数 >: このオーバーロードと \<filename > (\<行 >) のいずれか1つだけが \<paramname > に注釈が付けられています。または、どちらにも注釈を付ける必要があります。

この警告は、注釈にエラーがある場合に報告されます。
