# <a name="c28279"></a>C28279

> 警告 C28279: シンボルについて、'begin' が見つかりましたが対応する 'end' がありません

注釈言語は begin と end ( `{` および `}` C) コンストラクトをサポートしており、ペアリングが不均衡になっています。 マクロが使用されている場合は、この状況を回避できます。
