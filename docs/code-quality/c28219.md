# <a name="c28219"></a>C28219

> 警告 C28219: 注釈のパラメーターには、列挙型のメンバーが必要です

注釈のパラメーターは、名前付きの型のメンバーである必要があります。 **`enum`** また、他のいくつかのシンボルが検出されました。その型のメンバーを使用して **`enum`** ください。 これは通常、誤ってコード化された注釈マクロであることを示します。
