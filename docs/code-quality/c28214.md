# <a name="c28214"></a>C28214

> 警告 C28214: 属性パラメーター名は、p1...p9 である必要があります

この警告は、注釈の宣言を構築するときに、パラメーター名が p1... に制限されていることを示しています。p9. 通常の使用では、この警告は発生しません。
