# <a name="c28236"></a>C28236

> 警告 C28236: 注釈は式内で使用できません

この警告は、式コンテキストで注釈として宣言された関数を使用しようとしたことを示します。 注釈に標準のマクロが使用されている場合は、これを行うことはできません。この警告は、通常の使用では表示されません。
