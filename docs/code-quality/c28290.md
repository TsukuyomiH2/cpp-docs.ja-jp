# <a name="c28290"></a>C28290

> 警告 C28290: 関数に対する注釈には、実際のパラメーター数よりも多くの外部参照が含まれます

コード分析ツールは、関数の注釈に実際のパラメーター数よりも多くの外部参照が含まれている場合に、この警告を報告します。
