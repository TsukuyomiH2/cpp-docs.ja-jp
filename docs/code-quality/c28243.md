# <a name="c28243"></a>C28243

> 警告 C28243: パラメーターの関数の注釈には、注釈が付けられた実際の型よりも多くの逆参照が必要です

注釈の演算子の数 `__deref` が、パラメーターの型によって定義されたポインターのレベル数を超えています。 注釈の数値または参照されるパラメーターのポインターレベルを変更して、この警告を修正します。
