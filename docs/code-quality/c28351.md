# <a name="c28351"></a>C28351

> warning C28351: 注釈には、 \<annotation> 動的な値 (変数) を条件で使用できない場所が記述されています。

この警告は、C/c + + 型の検査で注釈が適用されたときに報告されます。

の式は、 `_When_` 定数に評価される必要があります。 `_When_`は無視されます。
