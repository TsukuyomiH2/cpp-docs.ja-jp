# <a name="c26462-use_const_pointer_for_variable"></a>C26462 USE_CONST_POINTER_FOR_VARIABLE

'% Variable% ' によってポイントされている値が1回だけ割り当てられています。 `const`へのポインターとしてマークしてください。 「 [ C++コアガイドライン con 4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction)」を参照してください。
