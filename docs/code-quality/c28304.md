# <a name="c28304"></a>C28304

> 警告 C28304: C++ 参照パラメーター <parameter_name> で、不適切 `_Notref_` に配置された演算子がに適用されました \<token> 。

演算子は、 `_Notref_` C++ 参照パラメーターを含む特殊な状況でのみ使用し、システム指定のマクロでのみ使用してください。 直後に演算子または機能的な注釈が続く必要があり `_Deref_` ます。
