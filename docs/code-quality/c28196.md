# <a name="c28196"></a>C28196

> 警告 C28196: 要件が満たされていません (式は true に評価されません)

この警告は、分析される関数の関数プロトタイプに、`_Out_` パラメーターまたは戻り値に `__notnull`、`__null` または `__drv_valueIs` があることを示しますが、返される値は、その注釈と一致しません。
